API Usage Examples

Replace YOUR_API_KEY with your actual API key.

Sending SMS via API

curl -X POST \
  '[API_BASE_URL]/api/sms' \
  -H 'Content-Type: application/json' \
  -H 'X-API-Key: YOUR_API_KEY' \
  -d '{
    "recipient": "+1234567890",
    "message": "Hello from the API!"
  }'

Getting Inbox Messages

curl -X GET \
  '[API_BASE_URL]/api/sms/inbox' \
  -H 'X-API-Key: YOUR_API_KEY'

Getting Outbox Messages

curl -X GET \
  '[API_BASE_URL]/api/sms/outbox' \
  -H 'X-API-Key: YOUR_API_KEY'

Getting System Logs

curl -X GET \
  '[API_BASE_URL]/api/logs' \
  -H 'X-API-Key: YOUR_API_KEY'
